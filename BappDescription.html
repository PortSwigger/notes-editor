The Notes Editor is a Burp Suite extension designed to enhance note-taking and report generation during security testing. The tool provides a rich text editor with tabbed documents, enabling easy navigation between notes. It comes with syntax highlighting to make writing and reviewing notes easier and more efficient. In addition, the extension also features built-in formatting tools tailored for penetration testers.

Features

  • Tabbed interface for multiple notes
  • Line numbers styled like VS Code
  • Header/Footer templates available for reports
  • Quick insertion of special symbols such as ?, ->, and ?
  • Notes can be saved/imported in .txt format
  • Logging HTTP request/response straight from Burp

Usage

The extensions interface is divided into three core components:

Tabbed Editor

  • Main Tab: always open and cannot be closed
  • Additional Tabs: also open but can be closed. A red X button appears on mouse hover.
  • Line Numbers: presented in a grey sidebar on the right side.

Button Panel

  • Format headers/footers
  • Save notes to a file
  • Clear the current tab
  • Highlight selected text
  • Load notes from a file
  • Duplicate the current tab
  • Insert special characters
  • Close the current tab (apart from the main tab).

Formatting Dialogs

  • Header/Footer Templates: Pre-formatted sections for report generation.
  • Symbol Palette: For quick insertion of Unicode symbols.

The extension also has the ability to log HTTP requests and responses from Burp directly into the notes. Commands for manipulating tabs, inserting special symbols, saving/reading to/from file, formatting documents and even sending HTTP requests/responses to the editor are all implemented for efficient note-taking and report generation.
